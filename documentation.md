# Labyrinths

**Версия:** 1.0.0  

Labyrinths — консольное приложение для генерации и решения лабиринтов. Поддерживает несколько алгоритмов генерации и решения, а также отображение лабиринта в консоли с использованием ASCII и Unicode-символов.

---

## Установка

1. Клонируйте репозиторий.

2. Постройте проект:

```bash
dotnet build
```

3. Запуск через CLI:

```bash
dotnet run --project ./src/Labyrinths.csproj -- [COMMAND] [OPTIONS]
```

---

## Запуск приложения

Общий формат:

```bash
dotnet run --project ./src/Labyrinths.csproj -- [COMMAND] [OPTIONS]
```

### Глобальные флаги

| Флаг         | Описание                              |
|--------------|---------------------------------------|
| `-H`, `--help`    | Показать справку                      |
| `-V`, `--version` | Показать версию приложения            |

## Команды

### `generate` — генерация лабиринта

Генерация лабиринта с заданным алгоритмом и размерами.

```bash
dotnet run --project ./src/Labyrinths.csproj -- generate -a dfs -w 21 -h 11 -o maze.txt
```

**Опции:**

| Флаг          | Параметр | Описание                                               |
|---------------|----------|--------------------------------------------------------|
| `-a`, `--algorithm` | string   | Алгоритм генерации: `dfs`, `prim`, `random`        |
| `-w`, `--width`     | int      | Ширина лабиринта                                    |
| `-h`, `--height`    | int      | Высота лабиринта                                    |
| `-o`, `--output`    | string   | Путь для сохранения файла                           |
| `-u`, `--unicode`   | bool     | Использовать Unicode-символы для отображения       |

---

### `solve` — решение лабиринта

Решение лабиринта с помощью выбранного алгоритма и вывод пути.

```bash
dotnet run --project ./src/Labyrinths.csproj -- solve -a dijkstra -f maze.txt -s 1,1 -e 9,9 -u true
```

**Опции:**

| Флаг          | Параметр | Описание                                               |
|---------------|----------|--------------------------------------------------------|
| `-a`, `--algorithm` | string   | Алгоритм решения: `astar`, `dijkstra`, `bfs`        |
| `-f`, `--file`      | string   | Путь к файлу с лабиринтом                            |
| `-s`, `--start`     | x,y      | Начальная точка лабиринта                             |
| `-e`, `--end`       | x,y      | Конечная точка лабиринта                              |
| `-o`, `--output`    | string   | Путь для сохранения решения                            |
| `-u`, `--unicode`   | bool     | Использовать Unicode-символы для отображения       |

---

## Алгоритмы генерации

1. **DFS (Depth-First Search)** — длинные извилистые пути с редкими развилками.  
2. **Prim** — более равномерно распределенные стены.  
3. **Random** — случайное заполнение с стенами и пустыми клетками.

---

## Алгоритмы решения

1. **A\*** — находит кратчайший путь с эвристикой Manhattan distance.  
2. **Dijkstra** — находит кратчайший путь без эвристики.
3. **BFS** — находит кротчайший путь без весов и эвристик.

## Символы для отображения

| CellType        | ASCII | Unicode |
|-----------------|-------|---------|
| Wall            | `#`   | `█`     |
| Empty           | ` `   | ` `     |
| Path            | `.`   | `•`     |
| Start           | `O`   | `▲`     |
| End             | `X`   | `▼`     |
