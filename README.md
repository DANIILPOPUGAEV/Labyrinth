Проект состоит из реализации консольной программы для генерации лабиринтов и поиска пути в них. Программа генерирует лабиринты различной сложности и размеров, а также предоставлено несколько методов поиска пути от заданной точки А (начала) к точке Б (конца).
Проект соответствует следующим функциональным требованиям:

[✓] Реализация 2-x алгоритмов генерации лабиринта методом поиска в глубину(DFS) и алгоритм Прима.
[✓] Реализация 2-х алгоритмов для поиска пути из заданной начальной точки O в конечную точку X: А* (A-star) и Дейкстры.
[✓] Ввод конфигурации лабиринта, начальной и конечной точки через CLI
[✓] Отображение пути через лабиринт при его наличии в консоль.
[✓] Сохранение решения в файл.
А также нефункциональным требованиям:
[✓] Код должен быть написан ясно и структурировано в соответствии с требованиями.
[✓] Программа должна иметь текстовый интерфейс.
[✓] Наличие документации по использованию, а также по алгоритмам генерации и решения лабиринта.
[✓] Должны быть покрыты основные тестовые сценарии: валидация, генерация и решение лабиринтов.

Дополнительно реализованы алгоритмы случайной генерации и алгоритм поиска в ширину. Также имеется реализация псевдографики с использованием unicode-символов, должна включаться отдельным флагом.
